## Project Description

The Peer Tutoring System is a prototype platform developed to connect students with tutors who can provide academic assistance. Through the system, students are able to request tutoring sessions, while tutors can review the requests and decide whether to accept or decline them. Administrators are responsible for monitoring users and managing system activities. The overall goal of the system is to make peer tutoring more organized, accessible, and efficient for students.
