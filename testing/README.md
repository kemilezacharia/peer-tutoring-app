# Testing Phase

During the testing phase, the prototype of the Peer Tutoring System was evaluated to ensure that the system functions correctly and that the different components interact as expected. The testing process included **unit testing** and **integration testing**.

## Unit Testing

Unit testing was conducted to verify individual components of the system. Each feature was tested independently to ensure it performs its intended function correctly. The following functions were tested:

* Student requesting a tutoring session
* Tutor accepting or rejecting a tutoring request
* Viewing tutoring session information
* Rating a completed tutoring session
* Admin generating system reports
